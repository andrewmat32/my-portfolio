01/About

A senior full-stack engineer with a designer's eye

I'm a Senior Full Stack Developer who builds web platforms end to end. For around five years I've shipped travel-booking platforms for tour operators and agencies across the Netherlands, Switzerland and Germany, owning everything from the Node.js back-end to the front-end UI. I'm grounded in core JavaScript and engineering fundamentals and fluent with frameworks and CMS (Vue, ApostropheCMS), reaching for the right tool for each job. Lately that includes AI-augmented development: I built and published a Claude AI + MCP code generator for ApostropheCMS. My background in Automation & Applied Informatics is the engineering discipline I bring to every build.

Today I'm the sole developer of the Victoury platform: a marketing CMS and a travel-booking minisite with a large library of custom widgets, built with an AI-augmented workflow. Before that I was a core developer on diving and safari sites like Diving World, building booking-engine features (pricing, cabins, flights and excursions), search, multi-step forms and reusable widget libraries.

I care about the fundamentals users feel but never see: accessibility (WCAG, automated Pa11y checks), performance, SEO and structured data, GDPR-compliant forms, and code other developers can maintain with confidence.

  • Design systems at scaleReusable widgets, tokens and themes that keep many sites consistent.
  • Performance-mindedLazy loading, lean token-driven CSS, fast first paint on any device.
  • SEO & structured dataJSON-LD, sitemaps and semantic markup that help products rank.
  • Accessibility-firstARIA, keyboard nav, contrast and reduced-motion, verified with a11y scans.

04/AI

Shipping with AI & MCP

Real, published AI tooling, not buzzwords: built on Claude AI and the Model Context Protocol.

Apostrophe Code Generator is an AI-powered module generator I built and published for ApostropheCMS. It uses Claude AI and an MCP (Model Context Protocol) server to generate production-ready pieces, pages, widgets and bundles (schemas, Nunjucks templates, design-token extraction and BEM SCSS) from a web wizard or directly inside Claude Code.

It's live on the official ApostropheCMS extensions marketplace, installable from npm, and indexed by MCP directories including LobeHub and mcpmarket. I bring the same AI-augmented workflow into client delivery, including the Victoury platform, pairing it with hand-written, production-grade code.

ApostropheCMS marketplace ↗ GitHub ↗ npm ↗
  • Claude AI + MCPAn MCP server exposing generation tools directly to Claude Code, plus a standalone web wizard.
  • Production-ready outputPieces, pages, widgets & bundles with schemas, templates, design tokens and BEM SCSS.
